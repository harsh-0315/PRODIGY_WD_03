# TIC TAC TOE Web APP

The Tic Tac Toe Web Game is an interactive, browser-based version of the classic 3x3 strategy game. Built using HTML, CSS, and JavaScript, it provides a clean and responsive interface where players can easily place their X’s and O’s. The game supports features like turn-based play, win detection, draw handling, and instant reset.

It offers smooth gameplay on all screen sizes and can be played in Player vs Player (PvP) or Player vs AI mode depending on the implementation. Designed for simplicity and fun, the app helps users improve their quick-thinking and strategy skills while enjoying a familiar game experience.
